HTML/CSS/JS Template

This is a basic HTML, CSS and JS template. It currently has:

  • Code for favicon and apple-touch-icons (iPhone/iPad inc. retina) - currently commented, use as required.
  • Conditional comments to add relevant classes for IE fixes.
  • Some media queries set to arbitrary break points.
  • JS detection and switch from Roger Johansson.
  • html5shim - https://code.google.com/p/html5shim/.
  • The iOS Orientation Change Fix (authored by @scottjehl, rebounded by @wilto).
  • Touch event handling through Hammer.JS with the jQuery helper to bind events.
